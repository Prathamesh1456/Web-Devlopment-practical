Study of Web Technology

1. Client Side (Front-End)

This is what the user sees in the browser.

Main tasks:
  • Page design
  • Buttons
  • Forms
  • Animations
  • User interaction

2. Server Side (Back-End)

This runs on the server.

Main tasks:
  • Processing user requests
  • Login system
  • Reading and storing data
  • Connecting database
  • API for AI project

3. Database

Stores:
  • Users
  • Books
  • Uploaded files
  • Questions
  • Answers

Different Web Technology Languages

1. HTML – HyperText Markup Language

Used to create the structure of a web page.

Examples:
  • Headings
  • Paragraphs
  • Forms
  • Images

It is the base of every website.

2. CSS – Cascading Style Sheets

Used for design and layout.

  • Colors
  • Fonts
  • Animations
  • Dark and light mode
  • Responsive design

3. JavaScript

Used for logic and interaction.

  • Button click
  • Form validation
  • API calling
  • Dynamic content loading

This is important for Upload, Ask, Summarize and Listen buttons.

4. Backend Languages

PHP
  • Form handling
  • Login system
  • Database operations
Python
  • Used with Flask and Django
  • AI projects
  • NLP
  • PDF processing
Java
  • Used with Spring and Spring Boot

5. Database Language – SQL

Used to store and retrieve data.

  • User records
  • Uploaded file information
  • History of questions

6. Web Frameworks

Frontend Frameworks
  • React
  • Angular
  • Vue
Backend Frameworks
  • Flask
  • Django
  • Express
